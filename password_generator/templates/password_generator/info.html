User's Guide


Password Generator usage steps:

On the Home page, there are 2 choices: Generate a random password and Test password security level.
On choosing 'Generate a random password', you will be shown some options to configure the password to be generated, like password length, whether to include uppercase letters, special characters and numbers, etc.
After choosing these options, i.e., your password generation criteria, and pressing Generate Password, your password will be displayed on the screen.
On reloading the page, another password will be generated using the same criteria.

On choosing 'test password security', you will be asked to enter a password. This password will not be stored anywhere or revealed anytime
On clicking Test security, the password's security level rating will be displayed along with a security score out of 28 points.
Security level ratings will be like 'Terrible!', 'Average', 'Very Low', 'High', 'Virtually Unhackable!', etc.

NOTE: This site is completely secure and safe to use. Your passwords will not be stored anywhere on the server, and cannot be accessed by anyone, not even us. The passwords entered by you to check security level will be deleted from the site completely once you leave that page, and no one can access them on the server.
Howerver, we recommend writing down these passwords as you may forget them.
